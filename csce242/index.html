CSCE 242: CLient-Server Computing

This class is a class about teaching client-server computing. Additionally this class will be about creating and designing a webpage about something of interest to me.

Assignments:

Assignment 1 - Basic HTML

This assignment was to get used to the basics of HTML including using images texts and creating lists.

Assignment 2 - Basic CSS

This assignment went more in depth into HTMl and focused on using CSS to style html sites and choosing color schemes.

Assignment 3 - Page Layout

This assignment tested students abilites to format and stye a page based on a given template. Also columns and flexboxes were introduced.
