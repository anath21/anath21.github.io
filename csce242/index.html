CSCE 242: CLient-Server Computing

This class is a class about teaching client-server computing. Additionally this class will be about creating and designing a webpage about something of interest to me.

Assignments:

Assignment 1 - Basic HTML

This assignment was to get used to the basics of HTML including using images texts and creating lists.

Assignment 2 - Basic CSS

This assignment went more in depth into HTMl and focused on using CSS to style html sites and choosing color schemes.

Assignment 3 - Page Layout

This assignment tested students abilites to format and stye a page based on a given template. Also columns and flexboxes were introduced.

Assignment 7 - Intro to JavaScript

This assignment was getting used to JavaScript and using it to change images, rotate images, and add stars to a box when the box is clicked.

Assignment 8 - If Statements

This assignment was an introduction to if statements where we had to change an image based on either a slider value or a letter entered by the user.

Assignment 9 - For Loops

This assignment was getting used to For Loops in JavaScript where we had to make a ball bounce when a button is clicked and had to display text when an image is clicked.

Assignment 10 - Arrays

This assignment was all about getting used to using arrays in JavaScript where we had to make advertisement text appear as a loop and also create an associated array of images of nature.

Assignment 11 - Classes

On this assignment we learned about classes and how to use the modal to make information appear on click. We also learned how to make class objects.

Assignment 12 - Parsing JSON

This assignment was all about parsing JSON and learning how to get data from a JSON file and make that display onto a html page.
